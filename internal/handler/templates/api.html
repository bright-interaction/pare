API och integrationer

MCP (AI-assistent)

Pares API är ett MCP-gränssnitt (JSON-RPC 2.0 över POST /mcp) med organisationsnyckel-autentisering. En AI-assistent som Claude kan läsa och bokföra åt dig. Kundernas namn, org.nr, personnummer och bankuppgifter tokeniseras vid gränssnittet (Shield) så att de aldrig lämnar programmet i klartext. Skrivningar har ett takbelopp och stöder dry_run (förhandsgranskning utan bokföring).
